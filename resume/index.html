這是一個基於 Gin Web Framework 建立的 web server架構，提供簡易的 http 與 websocket 路由。可透過 docker 容器啟動，若有動態擴展與自動部署需求可參照 cloudbuild.yaml 進行部署。

ELK

結合 reids & ELK 搜集分散式 log。適合想統一異質平台或已微服務化的系統使用。 僅需將目標 log RPUSH 至指定的 redis key，便可輕鬆地藉由 logstash 取出至 elasticserach 最後由 kibana 呈現。

WsTestTool

這是一個以 go語言編寫的 websocket壓測工具，提供併發連線、結果驗證、時間計算等功能。筆者目前使用的環境多為 Linux & MacOS，如果想在CLI簡單地對 websocket服務進行壓力測試的話，這邊提供一個簡單的工具給大家
