30Days ELK - Day23 Kibana Plugin X-Pack Install

January 7, 2017

今天要介紹一個超好用的 kibana plugin，X-Pack。你一定要安裝的好用插件讓我們先裝在說。

就讓我們照著官網的六步驟來安裝吧(只是我們是裝在 docker 上)

$ sudo docker exec -ti elasticsearch bash

$ cd /bin
$ elasticsearch-plugin install x-pack

  2. 於 kibana 安裝 X-Pack 同 elasticsearch 的部分，進入容器後安裝

$ cd /bin
$ kibana-plugin install x-pack
